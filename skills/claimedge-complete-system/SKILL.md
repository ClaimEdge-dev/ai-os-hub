# ClaimEdge Complete System v2.0

The single hub for ClaimEdge LLC. Every claims task routes through here; specialist skills (weather, materials, pricing, legal, investigation) plug in as spokes.

> **v2.0 consolidation:** Absorbs 10 skills — claim-command-center, claimed-master-workflow, and all 7 legacy exteriors skills (claims-master, claimedge-master, operations-suite, estimate-builder, supplement-writer, review-generator, brand-presentation-suite). The legacy pre-rebrand brand is retired — everything is ClaimEdge.

## The 5-Phase Claim Lifecycle (master workflow)

```
1. INTAKE & TRIAGE — loss details, policy pull, coverage flags, deadline clock starts
2. INSPECTION — Haag methodology, photo documentation protocol, material ID,
   hazardous-material flags (asbestos/lead), code violation capture
3. ESTIMATE — Xactimate scope with code upgrades, missed-line-item sweep (top 25),
   decking/ventilation/WRB arguments, Chicago 1.18x cost index
4. NEGOTIATION — carrier-ready supplement narrative, statutory rebuttals
   (215 ILCS 5/154.6, 5/155), carrier-specific counter-strategy, denial response
5. SETTLEMENT & CLOSEOUT — supplement payment tracking, depreciation recovery,
   review request, file archive, lessons to DECISION_LOG
```

## Standing Rules

- ClaimEdge branding on everything. No legacy brand references except archive context.
- Never assume storm dates, coverage, or code adoption — verify via the spoke skills.
- Every supplement line item gets a code or standard citation. No naked demands.
- Track deadlines from day 1: proof of loss, statute of limitations, appraisal windows.
