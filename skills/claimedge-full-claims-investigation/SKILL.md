# ClaimEdge Full Claims Investigation v2.0

The 12-agent litigation-grade investigation engine. Use when a claim needs maximum firepower — big supplements, denials, bad-faith setups, appraisal, or IDOI/litigation prep.

> **v2.0 consolidation:** Absorbs claims-evidence-orchestrator (same 12-agent architecture, attorney-ready packet format). One investigation engine, one packet standard.

## The 12 Agents (run in parallel, then synthesize)

1. **Property Records** — deeds, permits, assessor data, prior claims
2. **Meteorological** — NWS/NOAA/SPC storm verification for date of loss
3. **Building Codes** — IRC/IBC/IECC + local amendments supporting full scope
4. **Insurance Law** — 50 Ill. Adm. Code Part 919, 215 ILCS 5/155, relevant case law
5. **Field Documentation** — photo inventory, damage mapping, measurement verification
6. **Xactimate** — supplement line items, missed items, pricing verification
7. **Carrier Intelligence** — complaint history, tactic patterns, ratings
8. **Valuation** — ACV/RCV/O&L calculations, depreciation arguments
9. **Correspondence** — dispute letters with statutory citations
10. **Exhibit Packaging** — indexed, sourced, attorney-ready exhibits
11. **Pattern Analysis** — comparable claims, carrier underpayment patterns
12. **Red Team** — attack the packet: every objection the carrier will raise + counter

## Verification Standard (non-negotiable)

- Every claim of fact carries a source: URL, document, photo ID, or statute.
- Tag findings CONFIRMED / UNVERIFIED / NOT FOUND. Never fabricate evidence, storm data, or code citations.
- Contradictions between agents get flagged, not averaged.

## Output: Master Claims Packet

1. Executive summary (claim, loss, demand, strength)
2. Evidence matrix (fact → source → verification tag)
3. Code & statute appendix
4. Damage documentation index
5. Valuation breakdown (ACV/RCV/O&L)
6. Carrier correspondence drafts
7. Red-team report + rebuttals
8. Exhibit index (attorney-ready)
