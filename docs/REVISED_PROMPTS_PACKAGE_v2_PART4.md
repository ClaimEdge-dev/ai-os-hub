⚠️ **Permission Scope**: Comet requests access to view screens, send emails, access contacts, calendar. Start with minimal permissions.

⚠️ **Beta Software**: Occasional failures and hallucinations. Don't use for mission-critical tasks without verification.

⚠️ **Human-in-the-Loop**: For financial, legal, or credential-sensitive actions — always verify before confirming.
