# ClaimEdge LLC — Document Templates

**Independent Insurance Claims Inspection & Consulting | Chicagoland, IL**

This directory contains production-ready document templates for ClaimEdge LLC operations.

---

## Template Suites

### Storm Response Templates (`storm-response-templates.md`)
5 templates for storm event response and documentation:

| # | Template | Purpose |
|---|----------|---------|
| 1 | Storm Alert Email | Client notification after storm event |
| 2 | Storm Damage Inspection Checklist | Field inspection with Haag classification |
| 3 | Storm Event Documentation Log | Timeline and evidence tracking |
| 4 | Storm Supplement Narrative | Carrier-ready supplement with code citations |
| 5 | Client Storm Communication Status Update | Progress updates to insured |

**Legal Framework:** IRC 2021, IBC 2021, IECC 2021, 215 ILCS 5/155, 5/154.6, PA 103-0510

## File Structure

Due to GitHub API size limits, templates are split into parts:

```
storm-response-templates.md → storm_PART1.md + storm_PART2.md + storm_PART3.md + storm_PART4.md
commercial-claims-templates.md → commercial_PART1.md + commercial_PART2.md + commercial_PART3.md + commercial_PART4.md + commercial_PART5.md
```

To reconstruct the full file, concatenate parts in order.
